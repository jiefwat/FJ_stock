# Aster Market 今日研判链代码审查

## 状态

本地实现、独立代码审查、全量质量链、浏览器验收和公网发布验收均已完成。

## 当前范围

- 市场参与许可。
- 主线确认、候选、逆势异动、方向分歧和无主线状态。
- 研判链、结论证据、主线梯队与四工作区连续跳转。
- 首页无关模块、重复说明和失效样式清理。

## 当前 Findings

未发现未处理的高优先级问题。

- 收缩市场会把强势主题降级为“逆势异动”，不会误写成确认主线。
- 主线扫描和 `/api/opportunities` 复用同一市场许可，防守市场中的所有主题统一标记为“逆势异动”。
- 参与度、连续性或分歧字段缺失时保留不可用状态，主线判断降级为“方向分歧”。
- 外部快照文本仍经 HTML 转义，动态股票文本继续使用 `textContent`。
- 持仓数量和成本只进入 `localStorage`，网络请求只携带公开股票代码。
- 首页和页面 view 已删除旧地平线、事件流、全市场候选流及对应死代码，没有用隐藏模块伪装精简。

独立审查曾发现并已修复：缺失分歧字段被当成无分歧、防守许可未进入机会分类、旧页面字段仍留在 view，以及 1536px 沿用历史记录。相关问题均有回归测试或当前浏览器证据。

## 残余风险

- 参与许可与主线状态是确定性规则，不是预测模型，也不构成买卖建议。
- 页面结论质量取决于上游快照完整性；缺失值保留为不可用，不补演示数据。

## Decision

生产实时快照、统一研判语义、服务健康和唯一 release 均已验证，发布通过。
