# Aster Market 从零重建

## 目标

建立一个完全独立的公开只读 A 股市场态势应用，在单个桌面工作面回答市场状态、强势主题、观察候选和主要风险。

## 交付边界

- 新包名 `aster_market`，生产仅依赖 Python 标准库。
- 无旧代码祖先，不从其他分支合并组件或页面。
- 无侧栏、卡片仪表盘、深色终端、报告排版和手机代码。
- 无登录、账户、持仓、AI 问答、通知和交易接口。
- 公网首页可直接读取，动态数据明确标注时间与来源。

## 验收证据

- Pytest 覆盖模型、快照、推导、HTML 转义和 HTTP。
- Ruff 覆盖 `src/` 与 `tests/`。
- 浏览器覆盖 1280、1536、1920 三档桌面宽度。
- 公网检查 `/`、`/healthz` 和 `/api/snapshot`。
- 部署过程中不修改反向代理、旁路服务、定时器或原行情文件。
