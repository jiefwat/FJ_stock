# Aster Market 四模块分析甲板

## 目标

在现有市场地形上增加大盘分析、市场机会、股票分析和我的持仓，并保持标准库、公开只读、桌面专用和无交易边界。

## 交付

- 大盘：环境结论、四维证据、主要风险和下一观察点。
- 机会：主题阶段、强度、候选、证据和失效条件。
- 股票：搜索、趋势、动量、波动、估值、资金、事件和质量。
- 持仓：当前浏览器中的代码、数量、成本、市值、盈亏和主题暴露。

## 隐私

持仓只使用 `localStorage`，不读取旧持仓文件，不设置 Cookie，不通过 API 提交数量或成本。清除浏览器站点数据会同时清除持仓。

## 非目标

- 买卖、下单、目标价和自动交易
- 服务端持仓、账户和跨设备同步
- AI 对话和确定性投资建议
- 手机端和窄屏适配
