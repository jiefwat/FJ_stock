# 完整股票研究系统交付说明

## 交付内容

- 每日大盘：三指数、当日/区间趋势、主线、风险和快照时间。
- 我的持仓：当前账号前 20 只全量研究，当前数据为 11/11。
- 个股分析：八维结构化研究与缺口降级。
- 热点机会：每日 10 只候选、研究依据、失效条件和个股入口。
- 自动运行：07:20、12:10、18:30 三个北京时间检查点。

## 运行边界

- 公网必须登录；注册入口保留。
- 不把股数、成本、仓位、备注、Cookie 或用户标识发送到外部研究服务。
- 大盘与机会快照位于 `reports/research`；持仓和个股不落共享快照。
- 外部服务异常时可显示明确标记的历史快照，不伪装成实时结果。

## 部署目标

- 仓库：`/opt/stock-ts`
- Web 服务：`stock-ts.service`
- 每日任务：`stock-ts-daily-research.service` 与 `stock-ts-daily-research.timer`
- 公网：`https://stock.jiewat-kaka-fj.com/`

## 上线验收

部署后必须确认服务 active、timer active、两个快照成功生成、登录边界有效、四模块 HTTP 返回 200，以及本地/GitHub/服务器 `main` 哈希一致。
