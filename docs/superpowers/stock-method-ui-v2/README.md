# Stock Method UI V2

## 目标

把个股分析升级为“研究假设 -> 证据权重 -> 预期差与估值 -> 情景 -> 条件化动作 -> 失效”的专业决策链，并让页面在 30 秒内可读。

## 范围

- 专业个股档案模型与确定性生成规则。
- 研究假设、核心矛盾、因果链和加权证据。
- 研究条件与价格条件联动的仓位边界。
- 个股工作区桌面和移动端信息架构。

## 不做事项

- 不接交易执行。
- 不生成收益承诺或无依据目标价。
- 不修改用户持仓、生产数据和服务器配置。
