# 可用研究决策系统

## 目标

让四大原生研究工作台从“接口有返回”升级为“证据可解释、结论可复核、缺口不伪装”的可用系统。

## 规格与计划

- 设计：`../specs/2026-07-14-usable-research-decisions-design.md`
- 实施：`../plans/2026-07-14-usable-research-decisions.md`
- 验证：`test.md`

## 核心边界

- 每个能力使用独立的字段语义和最低证据要求。
- 身份、报价和内部元数据不能让研究维度通过质量门禁。
- 首屏只展示一条判断、一条动作、一个最大风险和最多三条不同发现。
- 缺失证据明确降级，不回退旧规则、不拼接无关结果、不补全不存在的结论。
