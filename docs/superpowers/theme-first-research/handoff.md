# 主题优先研究工作台交付说明

## 用户可见变化

- 每日大盘先展示结论、强势主题、真实涨跌分布和热门股票。
- 我的持仓按真实主题 token 聚合，只展示主要 8 个主题；分化区只保留至少 2 只持仓的可比主题。
- 个股分析输出“基本面承压 / 等待确认 / 可继续跟踪 / 证据不足”等人话结论，并写明原因与下一步。
- 热点机会先判断主线，再按命中的主题分组候选，展示确认条件和失效条件。
- 移动端删除背景旁白，首屏直接看到结论、行动、风险和主题。

## 数据与安全

- 大盘和机会使用每日快照；持仓和个股按登录用户实时研究。
- 持仓外部研究只提交代码、名称和已有主题，不提交股数、成本、权重或用户身份。
- 页面和公开 JSON 保持供应商中性；合法上市公司身份不会被品牌过滤误删。
- 所有动态节点使用安全 DOM API 和 `textContent`。
