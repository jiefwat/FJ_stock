# 主题优先研究工作台 Review

## Findings

未发现剩余 P0、P1 或 P2，当前改动可进入合并与部署。

审查过程中发现并关闭的主要问题包括：

- 缺行业持仓被静默漏掉、无主题候选被错误绑定主线。
- 负面事件、最终状态、低热度和市场宽度的语义误判。
- 多概念候选和多主题持仓无法按共同主题聚合。
- 供应商品牌过滤误删合法股票身份或生成伪分类。
- 科学计数、8 位金额与日期字段的格式歧义。
- 空宽度、负面决策颜色、候选分组和移动端可访问性问题。

上述问题均先补失败测试，再修复并通过复审。

## Open Questions / Assumptions

- 主题归属来自研究服务返回的概念和行业字段；缺失时明确进入“主题待补”。
- 持仓主题暴露按独立持仓数量统计，不使用股数、成本或权重。
- 模块级负面机会事件无法可靠映射到单只股票时，统一降级候选，不虚构个股归因。

## Residual Risks / Testing Gaps

- 外部研究能力的字段、限流和覆盖率可能变化，线上仍需持续观察缺口比例。
- 全仓仍有旧版 Web 契约和既有流水线环境失败；与 `main` 对比没有新增失败。
- 真实持仓的事件、机构预期和行情能力可能部分返回，页面会降级而不是补推测。

## Decision

Ready。研究引擎、公开协议、隐私边界、安全 DOM、桌面/移动响应式和真实数据链路均已验证。
