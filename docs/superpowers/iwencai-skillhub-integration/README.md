# 问财 SkillHub 研究集成

## 目标

在个股专业档案中增加一个精简、可审计的问财研究追问入口，通过服务端调用官方数据技能补足财务、机构预期、行业、事件、经营、股东、行情、公告、研报和新闻证据。

## 范围

- 问财 OpenAPI 客户端和技能路由。
- 同源个股研究 JSON endpoint。
- 个股页研究调度台及桌面、移动端交互。
- 无 Key、超时、限流、网关错误和空数据降级。

## 不做事项

- 不接真实交易或模拟交易。
- 不让社区技能直接影响本地评分和仓位。
- 不上传持仓、成本、Cookie 或账号信息。
- 不新增通用聊天历史和长期会话存储。
